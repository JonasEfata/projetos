### O que foi estudado hoje

**Leitura do `app.py` — conceitos aprendidos:**

| Conceito | Exemplo | O que significa |
|---|---|---|
| Importação | `from flask import Flask` | Buscar uma ferramenta de outro lugar |
| Variável | `app = Flask(__name__)` | Guardar um valor com um nome |
| Constante | `FRANQUIA_KM_MES = 5000` | Variável em maiúscula que não deve mudar |
| Ação/Função | `criar_tabelas()` | Parênteses = está executando algo |
| Rota | `@app.route('/')` | Define qual função responde a qual endereço |
| Definir função | `def index():` | Cria um bloco de ação com nome |
| Retorno | `return` | O que o servidor manda de volta ao navegador |
| GET | `methods=['GET']` | Navegador pedindo uma página |
| POST | `methods=['POST']` | Navegador enviando dados do formulário |
